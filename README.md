## E TİCARET APP

## kullandığım Teknolojiler

### - LiveData,
### - Coroutines
### - Retrofit ([Source](https://square.github.io/retrofit/))
### - Navigation Component(NavGraph, BottomNav)
### - Glide ([Source](https://github.com/bumptech/glide))
### - RecyclerView 
### - Fragments
